Sistema operativo:
Es el software principal que administra el hardware y los recursos del computador. 
Permite que las aplicaciones y servicios funcionen correctamente.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript en el servidor. Facilita la creación de aplicaciones web y APIs.

Aplicación server.js:
Es el archivo que contiene la lógica del servidor.
Define las rutas y las respuestas que recibirán los usuarios cuando hagan solicitudes.

Puerto:
Es un punto de comunicación utilizado por las aplicaciones para recibir conexiones. 
En esta práctica el servidor utiliza el puerto 3000.
Logs: Son registros que almacenan información sobre la actividad del sistema y de la aplicación. 
Ayudan a monitorear el funcionamiento y detectar errores.
